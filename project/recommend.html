Algorithm description

1. K-Means
- Unsupervised learning
- It studies data without label.
- It is often used to find the hidden structure in the data.
- Unsuvervised learning itself can be the purpose. (news, shopping)
- It can be the preprocessing step of Unsupervised learning.
- Clustering itself can be a label.

K-means clustering

- We want to divide the data into k clusters. K is arbitrary number that we chose.
- Centroid is the center of the each cluster. There are K centroids.
- Centroid can be chosen randomly in the data which we already have. It is not good to initialize all centroids to the same value.
- Normalization is needed, since it uses Euclidean distance. (If one variable is way too big then other variable cannot affect the result.)
- z normalization : (x-mean(x))/sigma
- min max normalization: (x- xmin)/ (x max – x min)

Process
1. Random 한 값으로 centroid 를 잡아준다.
2. 각각의 자료 값이 어떤 centroid 에 가까운지 그룹 지어준다.
3. 그룹끼리 mean 좌표를 다시 잡아준다.
4. 이 mean 값들과 자료 값들과의 거리를 다시 측정해서, 다시 그룹지어준다.
5. mean 값들이 변하지 않을 때까지 반복해준다.

MEMORY BASED CF:
